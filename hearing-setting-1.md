# 困難経験者向けアンケート設計資料

## **はじめに**
本資料は、職場や仕事に困難を抱え、精神的な問題や休職・退職といった状況に陥る若手社会人を対象としたサービス設計のためのヒアリング設計書です。対象者の現状やニーズを正確に把握し、課題に即した効果的なソリューションを提供することを目的としています。

---

## **背景**
近年、職場や仕事に起因するストレスやプレッシャーから、精神的な健康を損なう若手社会人が増加しています。特に10代後半から30代の世代では、以下のような傾向が指摘されています：

1. **主な困難**:
   - 職場での人間関係の問題（パワハラ、孤立など）
   - 長時間労働や過重労働による過労
   - 就職やキャリア形成への不安

2. **深刻化する問題**:
   - 精神疾患（うつ病、不安障害など）の発症
   - 休職や退職に追い込まれるケースの増加
   - 社会的孤立や自殺のリスク増加

---

## **目的**
本ヒアリングの目的は以下の通りです：

1. **困難の実態把握**:
   若手社会人が直面している具体的な課題や背景を理解する。
   
2. **立ち直りの要因特定**:
   困難を乗り越えた経験を分析し、効果的な支援策の要素を明らかにする。

3. **サービス開発の基盤形成**:
   本ヒアリングの結果を基に、匿名相談窓口やキャリア支援サービスなどのプロトタイプを作成し、実装へとつなげる予定です。

---

## **仮説**
1. 主な困難は「職場の人間関係の悪化」「過重労働」「経済的不安」である。
2. 支援を受けるきっかけは、信頼できる第三者（家族、友人、同僚）からの勧めが多い。
3. 困難を乗り越えた主な要因は「信頼できる人との対話」や「具体的な行動変化（例: 趣味や転職）」である。
4. 支援後も「孤立感」や「自己肯定感の低下」が残りやすく、フォローアップが不足している。
5. 若手社会人は「気軽に利用できる匿名の相談窓口」や「プライバシーが守られる支援環境」を求めている。

---

## **ヒアリング対象**
本ヒアリングの対象は以下の条件を満たす方とします：

1. **年齢層**: 10代後半から30代（ただし、困難を経験し克服した方は年齢を問いません）
2. **職業**: 正社員、契約社員、フリーランス、自営業、経営者、休職中、無職
3. **困難**: 職場や仕事で困難を経験し、それを克服した方（例: 精神疾患、休職、退職を含む）

---

## **質問項目**
### **1. あなたの基本情報**
- 年齢を教えてください。
  - 選択肢: 20歳未満 / 20～30代 / 40～50代 / 60歳以上
- 性別を教えてください。
  - 選択肢: 男性 / 女性 / その他 / 回答不可
- 現在の職業を教えてください。
  - 選択肢: 正社員 / 契約社員 / フリーター（アルバイト、パート） / 経営者・自営業・フリーランス / 無職 / その他

### **2. 過去に経験した困難**
- 当時の年齢を教えてください。
  - 選択肢: 10代後半 / 20代前半 / 20代後半 / 30代前半
- 当時の職業を教えてください。
  - 選択肢: 正社員 / 契約社員 / フリーター（アルバイト、パート） / 経営者・自営業・フリーランス / 無職 / その他
- 過去に直面した困難の内容を教えてください。（複数選択可）
  - 選択肢: 職場での人間関係の問題 / 過労や長時間労働 / 仕事のプレッシャーやストレス / 経済的な問題 / 家庭や友人との関係性の問題 / 精神的な問題（例: 不安、鬱など）/ その他
- その困難はどのくらい続きましたか？
  - 選択肢: 1ヶ月未満 / 1～6ヶ月 / 半年～1年 / 1～3年 / 3年以上
- その困難による精神的または身体的な影響があれば教えてください。（自由記述）
  - 例: 不眠、食欲不振、集中力低下、うつ病など

### **3. 休職や無職の経験**
- 困難を経験している間、休職または無職だった期間はありましたか？
  - 選択肢: はい / いいえ
- （「はい」と回答した場合）その期間はどのくらい続きましたか？
  - 選択肢: 1ヶ月未満 / 1～6ヶ月 / 半年～1年 / 1～3年 / 3年以上
- その期間中の生活や心境について教えてください。（自由記述）
  - 例: どのように日々を過ごしていたか、どのようなサポートや困難があったか

### **4. 立ち直りのきっかけや支え**
- 困難に対して取り組んだことがあれば教えてください。（複数選択可）
  - 選択肢: 家族や友人のサポートを受けた / 自分で問題解決の方法を探した / 趣味や活動を通じて気分転換を図った / 専門家（カウンセラー、医師）の支援を受けた / 教務支援窓口や制度を利用した / 環境を変えた（仕事、住居など）/ 自分を見つめ直す時間を作った / その他
- 立ち直る過程で役立ったことがあれば教えてください。（複数選択可）
  - 選択肢: 家族や友人のサポート / 趣味や活動 / 環境の変化 / 専門家の支援 / その他
- 立ち直りのきっかけとなった具体的なエピソードがあれば教えてください。（自由記述）

### **5. 提案と改善点**
- 「これがあればもっと早く立ち直れた」と感じるサポートや環境があれば教えてください。（自由記述）
- 今、あなたと同じような困難を経験している人に伝えたいアドバイスがあれば教えてください。（自由記述）

---

## **実施後のアクション**
1. **分析**:
   - 回答データをカテゴリ別に整理（例: 困難の背景、支援の効果、未解決の課題）。
   - 共通項を抽出し、ターゲット層のニーズを明確化。

2. **結果の活用方法**:
   - 支援プログラムの設計に反映（例: 気軽に相談できる窓口、段階的な復職支援）。
   - マーケティングや広報活動において、共感を得やすいメッセージを発信。
   - 他の支援者や団体と情報共有し、ネットワークの強化を図る。

3. **サービス提案**:
   - 具体例:
     - 匿名相談プラットフォームの構築
     - 定期的なフォローアップセッションの提供
     - キャリアやメンタルヘルスを支援するオンラインプログラム

---

## **おわりに**
本ヒアリングは、困難を経験している若手社会人の声を正確に把握し、適切な支援サービスを提供するための第一歩です。得られた結果をもとに、より多くの人々が孤立せずに前向きに生きられる社会を目指していきます。協力者や支援団体との連携を深めながら、サービス開発に活かしていく所存です。
